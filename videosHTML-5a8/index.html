Títulos e quebras de linhas

Sobre: Os títulos servem para dinamizar a leitura do texto grifando e mostrando as partes mais importantes de forma visual.
Já as quebras de linha auxiliam nessa função e também na organização de ideias.

Listas

Sobre: Servem para criar listas na página e são tags agregadas, pois, por exemplo, o ul precisa de pelo menos um li dentro dele pra se apresentar certinho.

Tipos:

  • ul: lista não ordenada
  • ol: lista ordenada (1, 2, 3,...)

Imagens e atributos HTML

Sobre: As tags de imagens servem para adicionar mídia no site, atuando junto das tags de vídeo e áudio, por exemplo.

Como adicionar no site (passo a passo):

  1. Adiciona a tag img
  2. Adiciona os atributos que você precisa...
    • QUAL imagem quer colocar, onde ela tá
    • Qual o TAMANHO da imagem
      • Usa-se o width (largura), se ajustar a largura ele automaticamente altera a altura (que seria o "height") proporcionalmente (e vice-versa).
    • Qual texto deverá aparecer se a imagem não o fizer e, mais importante, para facilitar os leitores de tela para portadores de deficiência visual.
  3. Prontinho, tá aí sua tag, agora vai lá testar se deu certo!

Sobre atributos: Os atributos especificam características das tags.
A sua anatomia consiste basicamente na tag, no nome e no valor do atributo.

Âncoras e links

Sobre: É muito útil pra trazer praticidade no acesso do conteúdo da página pelo usuário/cliente.

Dica extra: se usar o atributo target="_blank" na a href, vai abrir uma nova guia para o link.
